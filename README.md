Project Overview
This project analyzes user behavior and conversion performance using the Google Analytics 4 (GA4) public e-commerce dataset in BigQuery.
The analysis focuses on user sessions, conversion funnel progression, landing page performance, and engagement metrics to identify behavioral patterns affecting purchases.


Business Questions
- What percentage of users progress through the conversion funnel?
- Which traffic sources generate higher conversion rates?
- Which landing pages contribute most to purchases?
- Is there a relationship between user engagement and purchase behavior?

Analytical Scope
  funnel analysis
  landing page analysis
  engagement correlation

Tools Used
  BigQuery
  SQL
  GA4 Public Dataset
