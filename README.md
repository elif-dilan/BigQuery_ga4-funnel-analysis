Project Overview

Analyzed user progression across an e-commerce conversion funnel using GA4 event-level data in BigQuery.

Objective

Measure conversion and drop-off rates between key funnel stages to identify user behavior patterns and potential friction points.

Funnel Stages
  Session Start
  View Item
  Add to Cart
  Begin Checkout
  Purchase
  
Tools Used
  BigQuery
  SQL
  Google Analytics 4 (GA4)
  
Key Analysis
  Funnel conversion rates
  Drop-off analysis
  User progression tracking
  Event-level analytics
